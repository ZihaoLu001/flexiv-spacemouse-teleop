Owner Machine Setup

Installation Guide

This guide assumes the owner machine is the Ubuntu PC physically connected to the Flexiv control box, SpaceMouse, and optional camera. The recommended target is Ubuntu 22.04 with ROS 2 Humble.

Keep upstream Flexiv code separate. Clone this package beside flexiv_ros2 inside the same ~/teleop_ws/src workspace.

1. Install Owner Machine Dependencies

mkdir -p ~/teleop_ws/src
cd ~/teleop_ws/src
git clone https://github.com/ZihaoLu001/flexiv-spacemouse-teleop.git
cd flexiv-spacemouse-teleop

scripts/install_owner_machine_ubuntu22_humble.sh

The script installs ROS 2 Humble, MoveIt, MoveIt Servo, ros2_control, spacenavd, and the ROS spacenav package.

2. Fetch Flexiv ROS 2

cd ~/teleop_ws/src/flexiv-spacemouse-teleop
scripts/fetch_flexiv_ros2_humble_v1_7.sh

Use humble-v1.7 for Flexiv software package v3.9 and RDK v1.7 compatibility.

3. Build the Workspace

cd ~/teleop_ws/src/flexiv-spacemouse-teleop
scripts/build_workspace.sh

This builds Flexiv RDK dependencies into ~/rdk_install, then builds the ROS 2 workspace with colcon.

4. Source the Workspace

Add this to ~/.bashrc on the owner machine:

source /opt/ros/humble/setup.bash
source ~/teleop_ws/install/setup.bash

Open a new terminal and verify:

ros2 pkg list | grep flexiv_spacemouse_teleop
ros2 pkg executables flexiv_spacemouse_teleop

5. Run the Doctor

cd ~/teleop_ws/src/flexiv-spacemouse-teleop
export ROBOT_SN=Rizon4s-123456
export RIZON_TYPE=Rizon4s
scripts/doctor.sh

Replace ROBOT_SN with your lab robot serial before real hardware operation.
