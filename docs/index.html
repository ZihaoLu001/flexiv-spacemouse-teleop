Fast Path

Clone into a Flexiv ROS 2 workspace

The repo is a ROS package, not a monolithic robot image. Researchers can drop it into an existing `teleop_ws`, keep Flexiv upstream code separate, and tune the YAML parameters for each lab setup. 

mkdir -p ~/teleop_ws/src
cd ~/teleop_ws/src
git clone https://github.com/flexivrobotics/flexiv_ros2.git
git clone https://github.com/ZihaoLu001/flexiv-spacemouse-teleop.git
cd ~/teleop_ws
colcon build --symlink-install
